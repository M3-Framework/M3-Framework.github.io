Navcard

z-index: 2

Navcards are an alternative to the common navigation system.
They are intended to contain simple actions (as opposed to m3-nav).

You're Here Documentation About Support
You're Here Documentation About Support
You're Here Documentation About Support Features
Example:

<nav class="m3-navcard">
    <input type="checkbox" id="NavcardToggle">
    <section class="m3-navcard-content">
        <a class="m3-light-gray">You're Here</a>
        <a>Documentation</a>
        <a>About</a>
        <a>Support</a>
        <label class="m3-dropdown">
            <a class="dropdown-trigger">Dropdown</a>
            <input type="checkbox" class="m3-toggle">
            <div class="m3-dropdown-content">
                <a>One action</a>
                <a>Another action</a>
                <a>Something else</a>
            </div>
        </label>
    </section>
    <label for="NavcardToggle" class="m3-toggle m3-space-gray m3-huge">&equiv;</label>
</nav>
Empty Base:

<nav class="m3-navcard">
    <input type="checkbox" id="AnyID">
    <section class="m3-navcard-content">
        <a>Add More Actions</a>
    </section>
    <label for="AnyID" class="m3-toggle m3-huge">&equiv;</label>
</nav>
