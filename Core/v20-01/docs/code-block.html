Pure CSS
The code highlighter used in this documentation is not included in M3 Framework.

Code Block

Used to display large amounts of code in your site. 


fruits[0] = "Strawberry";
fruits[1] = "Pineapple";
fruits[2] = "Watermelon";

Inline code

Example: The html element defines an HTML page.

Use the class .m3-dark inside a code tag to change its background

Multiple lines of code:

<pre><code>
    /* Some code here */
</code></pre>
Single line of code:

<code>/* Some code here */</code>

Syntax Highlighting

You may use external libraries along with M3 Core to provide visual identification to your code.

Using highlight.js

CDN + Initialization
<link rel="stylesheet" href="http://cdn.jsdelivr.net/gh/highlightjs/cdn-release@9.17.1/build/styles/atelier-dune-dark.min.css">
<script src="http://cdn.jsdelivr.net/gh/highlightjs/cdn-release@9.17.1/build/highlight.min.js"></script>
<script>hljs.initHighlightingOnLoad();</script>

This library will automatically style any pre + code tags:


/* Place code between <pre><code> </code></pre> and it will be highlighted */

fruits[0] = "Strawberry";
fruits[1] = "Pineapple";
fruits[2] = "Watermelon";

Covering the full size of the code block:


<pre><code style="margin-left: 0; padding-left: 15px;">
    /* Some code here */
</code></pre>

Using W3 Code Color 

Source + Initialization
<script src="https://www.w3schools.com/lib/w3codecolor.js"></script>
<script>w3CodeColor();</script>

Use the classes .htmlHigh .cssHigh .jsHigh for code highlighting. 


/* JavaScript */

fruits[0] = "Strawberry";
fruits[1] = "Pineapple";
fruits[2] = "Watermelon";



/* CSS */

body {
    background-color: #d0e4fe;
}

h1 {
    color: orange;
    text-align: center;
}

p {
    font-family: "Times New Roman";
    font-size: 20px;
}



<!-- HTML -->

<h1> This is a heading </h1>
<p> And some text </p>
