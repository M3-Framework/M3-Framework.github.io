Pure CSS

Dropdown

z-index: 5

A dropdown allows the user to choose one value from a list. When a drop-down list is inactive, it displays a single value. When activated, it displays a list of values, from which the user may select one.

(You may use the class .dropdown-trigger when the trigger is not an .m3-btn)

Example:

<label class="m3-dropdown">
    <div class="m3-btn">Dropdown Btn</div>
    <input type="checkbox" class="m3-toggle">

    <ul class="m3-dropdown-content">
        <a>One action</a>
        <a>Another action</a>
        <a>Something else here</a>
        <hr>
        <a class="m3-text-skyblue">External Link 1</a>
        <a class="m3-text-skyblue">External Link 2</a>
    </ul>
</label>
Right-aligned Dropdown:

<div class="m3-dropdown-content" style='right:0;'>
    Your Content...
</div>

Hover Activation

As you might have noticed, we use a checkbox to toggle the content of a dropdown.
By using the onmouseover and onmouseout events, one may easily simulate a hover action: 


<label class="m3-dropdown">
    <div class="m3-btn"
    onmouseover="this.nextElementSibling.checked = true;"
    onmouseout="this.nextElementSibling.checked = false;">Hover me!</div>
    
    <input type="checkbox" class="m3-toggle">

    <div class="m3-dropdown-content">
        Some relevant information here...
    </div>
</label>
ALTERNATIVE (Using the :hover css selector) 

<label class="m3-dropdown hoverable">
    <div class="m3-btn">Hover me!</div>
    
    <input type="checkbox" class="m3-toggle">

    <div class="m3-dropdown-content">
        Some relevant information here...
    </div>
</label>

Inside Navigation

You can place dropdowns inside the navigation components

BRAND
You're Here Random 

<nav class="m3-nav m3-space-gray">
    <div class="m3-nav-brand">
        <a class="h4">BRAND</a>
        <span class="m3-nav-toggle" id="ToggleBTN">&equiv;</span>
    </div>
    <div class="m3-nav-section">
        <label class="m3-dropdown">

            <div class="m3-btn">Dropdown</div>

            <input type="checkbox" class="m3-toggle">

            <div class="m3-dropdown-content">
                <a>One action</a>
                <a>Another action</a>
                <a>Something else</a>
            </div>
        
        </label>        
    </div>
</nav>

Further Examples

Image Zoom

Click the image to display in full size


<label class="m3-dropdown">
    <img class="dropdown-trigger" src="some_img" width="40%" alt="Random Image"/>

    <input type="checkbox" class="m3-toggle">

    <div class="m3-dropdown-content">
        <img src="some_img" width="100%" alt="Random Image"/>
    </div>
</label>
Card Dropdown

Display cards inside dropdowns


<label class="m3-dropdown hoverable">
    <button class="m3-btn">Just a button</button>
    <input type="checkbox" class="m3-toggle">

    <div class="m3-dropdown-content">
        <section class="m3-card m3-shadow-2">
            <img src="some_img" class="m3-card-media">
            <article class="m3-card-content">
                Some text...
            </article>
        </section>
    </div>
</label>
