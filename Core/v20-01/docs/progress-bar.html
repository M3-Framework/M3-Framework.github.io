Progress Bars

A progress bar is a graphical control element used to visualize the progression of an extended computer operation, such as a download, file transfer, or installation.

Example:

<div class="m3-progress" style="height: 2rem;">
    <div class="m3-progress-bar m3-w-100" role="progressbar" aria-valuenow="100" aria-valuemin="0" aria-valuemax="100">
        <span class="m3-display-left m3-padding m3-large">SEXINESS üòè</span>
    </div>
</div>
Accessibility

The role and aria- attributes help clarify the status of the bar:

  • aria-valuemin: Minimum value.
  • aria-valuemax: Maximum value.
  • aria-valuenow: Current value.
Variable Height
Important:
Progress bars will not be displayed if there is no text inside the progress bar and/or the height is not specified.

Height: 1rem

Height: 3px

Change the height directly on the .m3-progress and the .m3-progress-bar will automatically resize. 


<div class="m3-progress" style="height: 1rem;">
    <div class="m3-progress-bar m3-w-75" role="progressbar" aria-valuenow="75" aria-valuemin="0" aria-valuemax="100"></div>
</div>

Vertical Progress Bar

Add the class .vertical to an .m3-progress to change its behaviour.
Now instead of modifying the width you will be modifying the height.

.m3-progress vertical 

<div class="m3-progress vertical m3-round-4x" style="height: 33vh;">
    <div class="m3-progress-bar m3-h-100" role="progressbar" aria-valuenow="100" aria-valuemin="0" aria-valuemax="100">
        <i class="fas fa-clock fa-2x m3-padding-2x m3-display-bottom"></i>
    </div>
</div>

Coloring

Customize an .m3-progress using M3 Colors or by manually setting the background and color.
This component is divided in two sections:

  • .m3-progress: represents the available space for the progress bar to grow in.
  • .m3-progress-bar: represents the actual filled amount of the progress component.
And therefore, you can modify these sections individually and create your own style.

TEXT
üêº 

<div class="m3-progress m3-black" style="height: 1rem;">
    <div class="m3-progress-bar m3-red m3-w-75" role="progressbar" aria-valuenow="75" aria-valuemin="0" aria-valuemax="100"></div>
</div>
Reminder: The colors used in this documentation are part of M3 Colors.
If you don't require our library, remember to replace each color with your own.
