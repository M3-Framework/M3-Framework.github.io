JavaScript:

function DisplaySidenav(id) {
    let sidenav_id = document.getElementById(id);
    sidenav_id.classList.toggle("active");
}
Display And Positioning Classes Available For M3 Sidenavs
Class Description
.m3-sidenav + .m3-right Displays the sidenav on the right corner of the page.
.m3-sidenav + .display-default Displays the sidenav on large screens (Desktop & TV) and hides it on smaller (Tablet & Mobile)
.m3-sidenav + .always-display Displays the sidenav always, regardless of the screen size.

Push Content #2

Content will be pushed to the right whenever the sidenav is active.

(Requires JavaScript)

HTML
<aside class="m3-sidenav m3-shadow-2" id="AnyID" style="width: 250px;">
    <a class="m3-option m3-light-gray">Option 1</a>
    <a class="m3-option">Option 2</a>
    <a class="m3-option">Option 3</a>
</aside>


<main id="main" style="transition: margin-left 0.6s;">
    <button class="m3-btn" onclick="ExPushContent('AnyID', 'main');">≡</button>
    ...
</main>


JavaScript
function ExPushContent(id, container) {
    let sidenav_id = document.getElementById(id);
    let container_id = document.getElementById(container);
    
    sidenav_id.classList.toggle("active");
    container_id.style.marginLeft = sidenav_id.style.width;

    if(!sidenav_id.classList.contains("active")){
        container_id.style.marginLeft = 0;
    }
}
