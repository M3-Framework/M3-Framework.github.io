Variants

Basic Container

.m3-container is a relative positioned container with a horizontal padding of 15px.
It's horizontally centered using margins.

This box should be horizontally centered.
But the text is not centered.


<section class="m3-container">
    Some content here...
</section>
Jumbo

.m3-jumbo has a considerable amount of padding and a bottom margin of 2em.
Like the basic container, the jumbo container has relative position.

This box should be horizontally centered.
I have 2em of horizontal padding and 3em of vertical padding.


<section class="m3-jumbo">
    Some content here...
</section>
Container Full Width

.m3-container-fw is a container with no paddings nor margins but with relative position. 


<section class="m3-container-fw">
    Some content here...
</section>
Padding & Margin

Using classes like .m3-padding or .m3-margin are an alternative to containers. Just keep in mind that the position will not be relative unless you combine containers with padding and margin classes. 


<section class="m3-container m3-padding-3x m3-vertical-margin">
    Some content here...
</section>

Full Screen

By using the class .m3-full-screen, your container's size will equal the size of the viewport
