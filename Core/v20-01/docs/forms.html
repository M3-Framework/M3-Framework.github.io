Checkbox

An styled checkbox is created by using a label tag with the class .m3-checkbox
They're not simple to customize, but they look better than an ordinary checkbox and have the same functionality.

Ordinary Checkbox (Boring) 


<label class="m3-checkbox"> 
    Some Text
    <input type="checkbox">
    <span class="checkmark"></span>
</label>
Further Styling

Unfortunately, customizing a checkbox is not as simple as customizing other components. You will need to change its styles directly. M3 Colors classes won't have any effect due to the selector specificity of this component.

Example:




<label class="m3-checkbox"> 
    Some Text
    <input type="checkbox">
    <span class="checkmark random-style"></span>
</label>

<style>
    .m3-checkbox .random-style { background: #dd973b;}  /* Box background color */
    .m3-checkbox .random-style::after { border-color: #ffbf0f;} /* Checkmark (‚úîÔ∏è) color */
    .m3-checkbox:hover input ~ .random-style { background-color: #cf7f7f;} /* Box background color on hover */
    .m3-checkbox input:checked ~ .random-style { background-color: #c44c4c;} /* Box background color when checked */
</style>

Radio

An styled radio is made in the same way as an styled checkbox.
The only differences are the checkmark sign and the roundness of the box.

Ordinary Radio (Boring) 


<label class="m3-radio"> 
    Some Text
    <input type="radio">
    <span class="checkmark"></span>
</label>
Further Styling

As expected, customization remains the same way for checkboxes and radios.

Example:




<label class="m3-radio">
    Some Text
    <input type="radio">
    <span class="checkmark random-style"></span>
</label>

<style>
    .m3-radio .random-style { background: #dd973b;}  /* Box background color */
    .m3-radio .random-style::after { border-color: #ffbf0f;} /* Checkmark (‚ô¶Ô∏è) color */
    .m3-radio:hover input ~ .random-style { background-color: #cf7f7f;} /* Box background color on hover */
    .m3-radio input:checked ~ .random-style { background-color: #c44c4c;} /* Box background color when checked */
</style>

Switch

A switch is basically a checkbox but with a different aspect.
You can toggle them on/off.

Dark Mode 

<label class="m3-switch"> 
    <input type="checkbox">
    <span class="slider"></span>
</label><span>Some Text</span>
Further Styling

A switch works in a similar way as that of a checkbox. However, switches use a slider instead of a checkmark.
M3 Colors classes won't have any effect due to the selector specificity of this component.

Example:

Pinky Pink Switch
Dark Side
Random Style 

<label class="m3-switch">
    <input type="checkbox">
    <span class="slider random-style"></span>
</label> <span>Random Style</span>

<style>
    .m3-switch .random-style { background: #dd973b;}  /* Box background color */
    .m3-switch .random-style::before { background-color: #ffbf0f;} /* Slider (‚ö™) color */
    .m3-switch input:checked ~ .random-style { background-color: #c44c4c;} /* Box background color when checked */
</style>
Reminder: The colors used in this documentation are part of M3 Colors.
If you don't require our library, remember to replace each color with your own.
