Alerts and Notifications

z-index: 10

Alert messages can be used to notify the user about something special: danger, success, information or warning.
An alert will remain in the page indefinitely but a notification is dismissed after 5 seconds.

HTML Only:

<div class="m3-alert m3-text-center">
    <button class="m3-close m3-small" onclick="this.parentElement.classList.add('m3-animate-bounce-out');">
        Close
    </button>
    <div class='m3-huge m3-margin-top-2x'>
        <span class='m3-icon'>üòé</span>
        <p>You are cool!</p>
    </div>
</div>
                            

<div class="m3-notification">
    <div class='m3-huge'>
        <span class='m3-icon'>üòé</span>
        <p>You are cool!</p>
    </div>
</div>

Where:

.m3-alert is the main container.
.m3-close positions itself to the top-right.

Using JavaScript:
Alerts

const m3_alert = document.createElement("div");
m3_alert.className = "m3-alert m3-text-center";
const close_btn = document.createElement("a");
close_btn.className = 'm3-close';

close_btn.innerText = "Close";
m3_alert.innerHTML += "Some elements...";
m3_alert.appendChild(close_btn);

close_btn.addEventListener("click", function(){close_btn.parentElement.classList.add('m3-animate-bounce-out');});
document.body.append(m3_alert);
                            
Notifications

const m3_notification = document.createElement("div");
m3_notification.className = "m3-notification m3-space-gray";
m3_notification.innerHTML += "You're cool! üòé";
document.body.append(m3_notification);

The following classes are available for box positioning:

Positioning Classes For M3 Alerts & Notifications
Class Description
.top-left Displays an .m3-alert/.m3-notification on the top-left corner
.top-right Displays an .m3-alert/.m3-notification on the top-right corner
.bottom-left Displays an .m3-alert/.m3-notification on the bottom-left corner

On small screens, an alert will be positioned in the bottom and will have a width of 100%. A notification, however, will remain unaffected and its position may be changed. Notifications are expected to have fewer content than alerts and therefore they will not obstruct user's screen.
