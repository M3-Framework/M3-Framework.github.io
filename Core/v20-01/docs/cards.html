Classes
Classes Used To Create M3 Cards
Class Description
.m3-card The main container. Images, header, body, and footer goes inside.
.m3-card-media This class is used in any kind of media that is displayed inside the card. (images, icons, etc...)
.m3-card-content The body of the card. If no media is displayed, you can still use this class for applying proper padding.
.m3-card-overlay Use this class instead of .m3-card-content to make the media element cover the whole card.

Cards are transparent by default, so make sure to add some color to them ;)

Overlay

You can replace the class .m3-card-content with .m3-card-overlay to make the media cover the whole card.

Round borders:
Use the class .round-overlay instead. 

<article class="m3-card">
    <img src="https://picsum.photos/640/360" class="m3-card-media" alt="Random Image">
    <div class="m3-card-overlay">
        <p class="m3-display-bottom m3-padding m3-w-100 m3-purple m3-small">
            Some text...
        </p>
    </div>
</article>
