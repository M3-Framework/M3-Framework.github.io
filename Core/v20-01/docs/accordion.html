    acc[i].addEventListener("click", function() {
        this.classList.toggle("active");

        } else {

        } 
    });
}

Where:

.m3-accordion is the element that shows/hides the content when clicked.
.m3-panel is the container where the content resides.

Active by Default

💡 You will need to identify your .m3-panel with the ID attribute.
IMPORTANT: Please make sure you put the ID in the .m3-panel ✔️ and not in the .m3-accordion ❌

Show me how! 

<div class="m3-accordion">I´m Active by Default!</div>
<div class="m3-panel" id="SomePanel">
    Some elements inside ... 
</div>

Copy this code, add the JavaScript below, and you are ready to go!

Just a few lines of JavaScript are required to make an accordion active:


let accpanel = document.getElementById("SomePanel"); /* Identify an specific panel to open */
accpanel.classList.toggle("active"); /* Optional: To keep track of whether or not an accordion is active.. */
accpanel.style.maxHeight = (accpanel.scrollHeight+5) + "px"; /* Makes the panel active (open); */
