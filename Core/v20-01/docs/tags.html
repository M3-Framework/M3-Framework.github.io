Tags - Badges (or Labels)

Tags can notify you that there are new or unread notifications. They can also be used to display cool messages, among other things. There is only one class provided for the creation of this component (badges, tags, labels. They all mean the same in here). You can make a tag look like a pill if you add the .m3-circle class to it.

Tags inside lists
  • We're number one! 7 New
  • I'm two but I have a very long name Hot!
  • Simply three Lol :)

<span class="m3-tag m3-circle">Hello!</span>
