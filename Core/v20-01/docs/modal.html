Modal Size

By default, modals have their min-width set to 35% and they will automatically expand themselves up to 85% width.
On mobile devices, a modal is set to have a width of 90%.

You can specify width measures for a modal using the sizing utilities: 


...
    <section class="m3-modal-content m3-w-75 m3-w-100-mobile">
        ...
    </section>
...

Bottom Sheet

Bottom Sheet Modals are useful for displaying actions to the user on the bottom of a screen.
They work in the same way as regular modals.

Show Bottom Sheet

Just add the class .bottom-sheet to your .m3-modal to make a Bottom Sheet Modal.

.m3-modal bottom-sheet

Adding JavaScript

Optionally, you can use javascript to automatically close the modal when the user clicks/touches outside the .m3-media-content

We use #Close to avoid going to the top of the page everytime a modal is closed.
Feel free to use any #Name as long as it doesn't belong to any html element in your page. 


var modal = document.getElementsByClassName("m3-modal");

var DismissModalEvent = function(event) {
    for(var i=0; i<modal.length; i++){
        if (event.target == modal[i]) {window.location.hash='#Close';}
    }
}

window.onclick = DismissModalEvent;
window.ontouchend = DismissModalEvent;
