Buttons

A button is a fundamental UI element that will heavily affect your interaction design. Buttons have the power to compel users to convert, to carry out an action.

Default Style

By default, a <button> will look like this:

M3 Button

Give some style to your button by adding the class .m3-btn

Result:

Link

Add the class .no-margin to an .m3-btn to take out its margins.

class="m3-btn no-margin"
Bordered Button

Add the class .m3-transparent to take out the background. Then proceed to add a border using the class .m3-border or use the class .m3-bar for thick borders.

Result:

Round Buttons

Add the class .m3-round or .m3-circle to apply roundness.

Result:

You can give some life to these buttons using M3 Colors

Default button:

<button>I'm transparent and sexy!</button>
Basic button:

<button class="m3-btn">Button</button>
<a class="m3-btn">Link</a>
Bordered button:

<button class="m3-btn m3-transparent m3-border">Button</button>
<button class="m3-btn m3-transparent m3-bar"><b>Button with thick border</b></button>
Round button:

<button class="m3-btn m3-round">Round Button</button>
<button class="m3-btn m3-circle">❤️</button>
Buttons using M3 Colors:

<button class="m3-btn m3-orange">Orange</button>
<button class="m3-btn m3-purple">Purple</button>
<button class="m3-btn m3-teal">Teal</button>
<button class="m3-btn m3-blue-gray">Blue Gray</button>

<button class="m3-btn m3-transparent m3-border m3-border-orange m3-text-orange">Orange</button>
<button class="m3-btn m3-transparent m3-border m3-border-purple m3-text-purple">Purple</button>
<button class="m3-btn m3-transparent m3-border m3-border-teal m3-text-teal">Teal</button>
<button class="m3-btn m3-transparent m3-border m3-border-blue-gray m3-text-blue-gray">Blue Gray</button>

<button class="m3-btn m3-hover-orange">Hover Orange</button>
<button class="m3-btn m3-hover-purple">Hover Purple</button>
<button class="m3-btn m3-hover-teal">Hover Teal</button>
<button class="m3-btn m3-hover-blue-gray">Hover Blue Gray</button>

Changing Size

Use any of the .m3-size classes to change the size of a button


<button class="m3-btn m3-tiny">Tiny</button>
<button class="m3-btn m3-small">Small</button>
<button class="m3-btn m3-large">Large</button>
<button class="m3-btn m3-huge">Huge</button>

Button Group

Wrap your buttons inside a .m3-btn-group container and it will look like this:


<div class="m3-btn-group">
    <button class="m3-btn">Button 1</button>
    <button class="m3-btn">Button 2</button>
    <button class="m3-btn">Button 3</button>
    <button class="m3-btn">Button 4</button>
</div>
Reminder: The colors used in this documentation are part of M3 Colors.
If you don't require our library, remember to replace each color with your own.
