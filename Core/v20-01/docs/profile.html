Pure CSS

Profile

Identify users and use indicators to display their current status.

Image Profile:

<figure class="m3-profile">
    <img src="your_img" alt="Name or Description">
    <span class="m3-status online"></span>
</figure>
Text Profile:

<figure class="m3-profile" profile-text="XD">
    <span class="m3-status online"></span>
</figure>
Profile with Icon Inside:
Option 1
<figure class="m3-profile" profile-text="XD">
    <div class="m3-icon m3-black">
        <span class="fas fa-globe-asia"></span>
    </div>
</figure>


Option 2
<figure class="m3-profile">
    <img src="your_img" alt="Name or Description">
    <img src="your_icon" class="m3-icon">
</figure>

Indicators

  • m3-status .online: Represents connection / presence.
  • m3-status .unavailable: Equivalent to being busy / away.
  • m3-status .offline: Represents disconnection / absence.


<figure class="m3-profile" profile-text="ON">
    <span class="m3-status online"></span>
</figure>

<figure class="m3-profile" profile-text="UA">
    <span class="m3-status online"></span>
</figure>

<figure class="m3-profile" profile-text="OF">
    <span class="m3-status online"></span>
</figure>

Sizing Utilities in M3 Profiles
Size Details
.m3-tiny width & height: 2rem.
.m3-small width & height: 3rem;
Default width & height: 4.8rem;
.m3-large width & height: 7.2rem;
.m3-huge width & height: 9.6rem;
- If you will manually resize a profile, remember to modify its font-size along with the width and height.
Reminder: The colors used in this documentation are part of M3 Colors.
If you don't require our library, remember to replace each color with your own.
